This application is in beta-version, it is possible that some calculations could be inaccurate, so if you find any bug or error please let us know contact@gene-calc.pl

The Gene-Calc is a constantly developed tool dedicated for calculations related to biological sciences, especially focused in field of genetics. Application offers:
  • testing of compliance with the Hardy-Weinberg model
  • tests from the chi-square family with coefficients of correlation, for contingency tables of any size
  • determination of Polymorphic Information Content (PIC) and Heterozygosity (H) values for both codominant and dominant markers
  • estimate genetic distance
